請注意，對於新任務，如果您還沒有看到先靈提供的記憶，先靈會要求您在解鎖其每日任務之前重溫他們的記憶。在這種情況下，地圖神壇會告訴你在哪裡可以做到這一點！

## 新功能：收藏表情
您可以在光•遇中使用數十種表情來表達自己，在 0.26.5 中，我們很高興推出一項功能，讓您更輕鬆地使用您最喜歡的表情！

透過幾種不同的方式存取這個新的收藏夾選單：
- 在一次性教學的主表達式動作表情選單中（稍後會詳細介紹）
- 按住 光翼能量計
- 按鍵盤上的 Z 或按滑鼠中鍵
- 按控制器/手把/搖桿/手柄上的 L2。

這將顯示一個輪盤選單，其中包含表達式動作表情選單中八個操作的插槽，您將能夠新增或刪除插槽所需的任何表達式動作表情（或更多）。

下載此更新後首次登入光•遇時，「表達式動作表情」選單中將出現一個星形按鈕。點擊它即可快速了解如何使用該功能。完成教程後它將消失。（對於全新玩家，一旦他們到達禁閣就會出現。）

此功能應該可以讓您更輕鬆地發送您最喜歡的表情、準備您最好的舞蹈動作，甚至編排您的下一個共享記憶！

## 新功能：在友誼星座解除好友關係/從好友星座&星盤上解除好友關係
友誼星座記錄了我們在光•遇中建立的所有聯繫。但有時，能夠調整我們的星座是有幫助的，特別是當我們與一些光之子一起玩的次數比以前少時，或者如果我們希望能夠歡迎新朋友加入擁擠的星座。

此更新引入了一項功能，允許玩家在友誼星座中查看某人的光之子時與某人解除好友關係。例如，這與封鎖的工作方式不同，封鎖被封鎖的玩家無法與封鎖他們的人進行互動。

取消好友關係：
- 刪除好友的玩家將從您的星座/星盤中刪除。
- 您和對方之間的所有好友關係能力都被禁用（包括聊天）。
- 有可能會遇到一個你刪除好友的人。如果發生這種情況，您和未加好友的玩家在遊戲中再次見面時將顯示為陌生人。
- 如果你點亮他們的光之子，他們的頭上就會出現一個圖標，表明他們是你刪除好友的人。
如果您遇到了不再加為好友的人，您都可以選擇重新點燃友誼，而無需額外花費蠟燭。所有先前解鎖的關係能力都將恢復。

## 其他更新
- 自動跳過過場動畫開關：自動跳過過場動畫的選項已新增至畫質選項下的設定選單。啟用後，只要玩家之前至少看過一次可跳過的過場動畫，就會自動跳過它們。

無法自動跳過的過場動畫通常會導致世界發生物理變化，例如開門。在這種情況下，玩家將在門根據正常物理定律打開時保持對攝影機的控制，而不是從一幀到下一幀閃爍打開。 

如果您對自動跳過過場動畫開關有任何回饋，請在 Discord 上的 #live-feedback 頻道中與我們分享，或在我們的官方 Facebook 社團中的回饋貼文討論串中發表評論！

- 該團隊最近開始致力於一個項目，該項目最終將使我們能夠更快地為玩家提供更新和修復。這仍在進行中，但在某些情況下，啟動遊戲時可能會出現此訊息：“最新內容下載中。”這與團隊正在開發的專案有關，所以這不是一個錯誤。

## 此更新中的改進和錯誤修復包括
### 強調
- 作為每日任務更新的一部分，任務進度狀態更新現在將在過場動畫中顯示（如果適用）。所有每日任務的完整清單將保持隱藏。
- 先靈在樹林高處上給予的光之饅頭已經回到地面，以便光之子可以輕鬆地再次收集它們。
- 使用控制器/手把/搖桿/手柄時，現在好友樹選單和握住朋友的手或跟隨他們的按鈕提示優先於其他互動按鈕提示。
- 現在，被牽手時打開雨傘應該可以保護被牽手的玩家和主牽手他們的人免受雨水傷害。
- 使用風之大道到達隱士山谷應該不再將你彈回風之大道。
### 遊戲玩法
- 修正了在水下被肩背的掛機狀態玩家時會耗盡呼吸計和能量的錯誤。
- 在禁閣中使用控制器/手把/搖桿/手柄應不會再導致玩家沉到地板下。
- 修正了啟動即興演奏台時在和聲大廳中可能發生的閃退問題。
- 音樂可以非常令人著迷，但又不會太令人著迷，以至於會導致像阻止黑暗螃蟹攻擊坐著演奏樂器的玩家那樣的錯誤，因此該錯誤已得到修復。
- 當全螢幕選單可見時，千人房系統模式下的反應選單不會顯示。
- 修正了一個錯誤，該錯誤可能會阻止玩家重命名最近新增的朋友，或無法正確使用牽手、肩背和搖籃抱來攜帶互動。
- 當您與在查看玩家創作的 UI 時，它們的朋友一起/"在肩膀上"，應該會出現用於查看玩家創作的 UI。
- 將二重奏季樂譜中的低音部分更新為高八度，以改善演奏時的音樂效果。
- 對於新玩家：在入口暢通之前，雲巢魔法商店應該無法再進入。
- 當雲野神廟/神殿的長老過場動畫後的提示文字顯示如何進行大呼叫時，進行常規（小）呼叫將不再繞過教程並可能誤導新玩家。
- 另一個可能同樣影響新玩家的錯誤：修復了大呼叫教程中的一個錯誤，該錯誤向新玩家展示了蝴蝶如何幫助他們飛行，如果玩家打開了表情選單，則允許玩家移動。
### 化妝品
- 二重奏嚮導的西裝燕尾服跪著也不會顯得變形。
- 解決了 SkyFest 線框紋斗篷與角色頸部的夾角問題。
